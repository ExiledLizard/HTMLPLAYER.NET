A Hidden Feature (Text Box)

Here It Is

So, you ended up clicking the Info SVG image from curiosity...? Well, the hidden feature of the Text Box, is that there are placeholders. How these "placeholders" work, is that they replace a specific keyword, to its meaning. Here's an example sentence with a placeholder in action, using the {time} placeholder: Hello, Santa, the current time is: 20:01. Here are all of the available placeholders that the Input Box supports:

No .1: {fulldate} - Example output: Thu Dec 05 2024 18:51:08 GMT-0800 (Pacific Standard Time).

No .2: {date} - Example output: 04/16/2008.

No .3: {year} - Example output: 2008.

No .4: {month} - Example output: 04.

No .5: {dayofmonth} - Example output: 26.

No .6: {time} - Example output: 21:03.
