Random Number Generator

Welcome to the Random Generator page of htmlplayer.net! What this page allows you to do, is to simply generate any random number not greater than a user-specified value! To use this page, enter any number in an input box. Then, press the Evaluate button (under input box), and you'll get a random number not greater than the one you input.
