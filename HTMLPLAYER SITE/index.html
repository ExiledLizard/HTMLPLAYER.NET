Credits

Thanks to the following CDNs, that make my site's animations, CSS, and other features a thing (!):

No 1. Bootstrap

Bootstrap is a modern CDN, allowing developers to make their site's styling better, add components like this menu (tooltips, dropdowns, modals, e.t.c) to their UIs, and more. Click the image, or link in this text, to get redirected to this CDN's site.

No 2. Bulma

Bulma is like Bootstrap, but it's animations, features, and other aspects are different. Click the image, or link in this text, to redirect to Bulma's site.

No 3. Popper

Popper, is a CDN that allows developers to add many types of toolips, popovers, floating UIs, and e.t.c to their sites' elements (for example, a little square appears, containing developer-specified information about an element your mouse is hovered over). Click the image, or link in this text to redirect to their main site.

No 4. Ace (The Editor Window)

Ace, is a embeddable (via CDN, e.t.c) web-based code editor, that supports many programming languages (30+). It is very customizable (programming language, theme, e.t.c) with JavaScript. Click the image, or link in this text, to be redirected to the CDN's site.

Introduction

Welcome to this site! What this site allows you to do, is create websites like this one, using whatever inputted HTML code there is in the Editor (the Top Box). The code from the Editor gets transfered into the Player (the Bottom Box), and is displayed as HTML elements.

.

Usage

To use this site, you simply need to input HTML code into the Editor. Once you input valid HTML code into the Editor, the outputted HTML elements will show in the Player. Don't know how to code in HTML? Click here, for a HTML tutorial - happy coding!
